# Copypaste
A website where you can copy and paste quickly without login and ads (has captcha)

You can share text in real time via websockets and share quickly text from different devices without using some other apps or services that were not designed for it

Code almost entirely made by me (both for front and backend)
## Live website at https://copypaste.filippodude.com/
